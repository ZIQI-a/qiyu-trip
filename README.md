# 奇遇旅途 | 移动端租房项目

## 项目介绍

本项目是基于Vite + Pinia + Vant 3 编写的模仿某民宿网站的移动端页面。

## 项目启动

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
